Commenter Service

The Commenter package provides support for extensions to support the Comment and Uncomment commands in Visual Studio using MEF.

Library Status: Under Development (not yet released)

Installation and Distribution

This package is available through NuGet under the following package IDs.

  • Tvl.VisualStudio.Commenter: This package should be installed into the project that creates your VSIX file. If your Visual Studio extension is not broken into multiple sub-projects, this is the only Output Window package you will need to explicitly install to use these features.
  • Tvl.VisualStudio.Commenter.Interfaces: This package should be installed for any project that uses the Commenter Service.

After installing the appropriate NuGet package(s), you will need to configure your extension to inform Visual Studio of the dependency by including the following block in your source.extension.vsixmanifest file.
